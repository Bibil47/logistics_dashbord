## Project Overview

This project focuses on analyzing logistics and inventory management data to evaluate warehouse performance, inventory efficiency, and demand trends. The analysis was performed using Python for data preprocessing and machine learning, while Power BI was used to create an interactive dashboard for business insights.

## Objectives

* Analyze inventory and warehouse operations.
* Monitor stock levels and stockout incidents.
* Evaluate logistics performance using KPIs.
* Identify demand patterns across product categories.
* Forecast future demand using machine learning models.
* Develop an interactive Power BI dashboard for decision-making.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Power BI
* VS Code

## Dataset Features

The dataset includes:

* Stock Level
* Daily Demand
* Reorder Point
* Lead Time
* KPI Score
* Order Fulfillment Rate
* Picking Time
* Turnover Ratio
* Stockout Count
* Forecasted Demand

## Data Preprocessing

* Removed duplicate records
* Checked missing values
* Validated data consistency
* Formatted date columns
* Prepared data for analysis and modeling

## Machine Learning

Multiple regression models were evaluated:

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor
* Gradient Boosting Regressor

### Best Model

**Random Forest Regressor** achieved the best predictive performance and was selected as the final forecasting model.

## Power BI Dashboard

The dashboard includes:

* Total Stock
* Total Orders
* KPI Score
* Average Daily Demand
* Total Stockouts
* Stock Level by Category
* KPI Score by Zone
* Fulfillment Rate by Zone
* Average Picking Time by Zone
* Turnover Ratio Analysis

## Conclusion

This project demonstrates how data analytics, machine learning, and business intelligence tools can be used to improve inventory management, demand forecasting, and warehouse performance monitoring.
